# Módulo 7 - Docker

## Objetivo

Aprender a contenerizar aplicaciones Spring Boot para facilitar su despliegue y ejecución.

## Temas

- ¿Qué es Docker?
- Docker Engine
- Imágenes
- Contenedores
- Dockerfile
- Docker Compose
- Volúmenes
- Redes
- Variables de entorno
- Multi-stage Build
- Optimización de imágenes

## Proyecto

Dockerizar:

- ms-clientes
- MySQL
- API Gateway
- Eureka
- Config Server

## Reglas

- Un Dockerfile por microservicio.
- No almacenar credenciales en la imagen.
- Usar variables de entorno.
- Utilizar Docker Compose para desarrollo.
- Mantener imágenes ligeras.

## Al finalizar

El estudiante será capaz de ejecutar el sistema bancario completo mediante Docker Compose.
