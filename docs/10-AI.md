# Módulo 10 - Entrevistas Java Backend

## Objetivo

Preparar al estudiante para superar entrevistas técnicas de Java Backend y Spring Boot mediante simulaciones, retos y defensa del proyecto.

## Temas

- Preguntas de Java
- Preguntas de POO
- Preguntas de Collections
- Streams
- SQL
- Spring Boot
- Spring Security
- JPA
- Hibernate
- Testing
- Docker
- Microservicios
- Git
- Arquitectura
- Optimización
- Debugging
- System Design (Introducción)

## Proyecto

Defensa técnica del Sistema Bancario Enterprise.

El estudiante deberá justificar:

- Arquitectura elegida.
- Diseño de la base de datos.
- Uso de DTOs.
- Validaciones.
- Manejo de errores.
- Estrategia de testing.
- Seguridad.
- Dockerización.
- División en microservicios.

## Simulaciones

- Entrevista técnica.
- Code Review.
- Debugging.
- Reto Backend.
- Refactorización.
- Diseño de API REST.

## Reglas

- Explicar antes de responder.
- Justificar las decisiones técnicas.
- Pensar como un desarrollador Backend profesional.
- Relacionar las respuestas con el proyecto bancario.

## Al finalizar

El estudiante será capaz de defender técnicamente un proyecto Java Backend y afrontar entrevistas con confianza.
