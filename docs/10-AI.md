# Módulo 10 - IA Aplicada al Desarrollo Backend

## Objetivo

Aprender a utilizar Inteligencia Artificial como herramienta para aumentar la productividad sin reemplazar el criterio del desarrollador.

---

# Herramientas

- ChatGPT
- Google AI Studio
- Gemini
- GitHub Copilot
- Claude

---

# Casos de uso

- Generación de código.
- Explicación de errores.
- Refactorización.
- Documentación.
- Testing.
- SQL.
- Docker.
- Microservicios.
- Code Review.
- Optimización.

---

# Buenas Prácticas

Utilizar IA para:

- Aprender.
- Revisar código.
- Detectar errores.
- Generar ejemplos.
- Crear pruebas.

No utilizar IA para:

- Copiar código sin comprenderlo.
- Resolver entrevistas.
- Generar proyectos completos sin analizarlos.
- Ignorar buenas prácticas.

- Ejecutar pruebas antes de aceptar cambios.
- Comprender el código generado.
- No compartir credenciales.
