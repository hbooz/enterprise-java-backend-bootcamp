# Módulo 1 - Java Profesional

## Objetivo

Formar una base sólida en Java para desarrollar aplicaciones Backend con Spring Boot.

## Temas

- Variables
- Tipos de datos
- Operadores
- Condicionales
- Bucles
- Métodos
- Clases
- Objetos
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción
- Interfaces
- Excepciones
- Generics
- Collections
- Streams
- Optional
- Lambdas
- Records

## Proyecto

Todo el módulo se desarrollará sobre un Sistema Bancario.

Entidades:

- Cliente
- Cuenta
- Transacción

## Al finalizar

El estudiante será capaz de construir un modelo orientado a objetos profesional antes de comenzar Spring Boot.
