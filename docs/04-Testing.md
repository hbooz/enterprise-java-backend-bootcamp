# Módulo 4 - Testing Profesional

## Objetivo

Aprender a validar la calidad del código mediante pruebas automatizadas.

## Temas

- Testing Unitario
- Testing de Integración
- JUnit 5
- Mockito
- MockMvc
- Assertions
- Mocks
- Stubs
- Verify
- JaCoCo
- Cobertura de código
- Test Pyramid
- Buenas prácticas de testing

## Proyecto

Pruebas del sistema bancario:

- ClienteService
- CuentaService
- TransaccionService

## Reglas

- Todo Service debe tener pruebas unitarias.
- Todo Controller debe probarse con MockMvc.
- No probar el Repository directamente.
- Cubrir casos exitosos y casos de error.
- Objetivo de cobertura: 80% o superior.

## Al finalizar

El estudiante será capaz de escribir pruebas profesionales para aplicaciones Spring Boot.
