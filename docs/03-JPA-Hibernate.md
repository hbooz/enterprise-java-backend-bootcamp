# Módulo 3 - JPA e Hibernate

## Objetivo

Aprender a persistir datos de forma profesional utilizando Spring Data JPA e Hibernate.

## Temas

- ORM
- Entity
- Repository
- CrudRepository
- JpaRepository
- Relaciones OneToOne
- OneToMany
- ManyToOne
- ManyToMany
- Cascade
- FetchType
- Lazy Loading
- Eager Loading
- JPQL
- Native Query
- DTO Projection
- Pagination
- Sorting
- Specifications
- EntityGraph
- Optimización de consultas
- Evitar N+1

## Proyecto

Persistencia del sistema bancario:

- Cliente
- Cuenta
- Transacción

## Buenas prácticas

- Nunca exponer Entity en la API.
- Usar DTO.
- Minimizar consultas a la base de datos.
- Evitar N+1.
- Usar paginación.
- Aplicar transacciones correctamente.

## Al finalizar

El estudiante será capaz de diseñar una capa de persistencia profesional y optimizada.
