# Módulo 11 - Entrevistas Java Backend

## Objetivo

Preparar al estudiante para entrevistas técnicas de Java Backend, Spring Boot, microservicios y arquitectura empresarial.

## Temas

- Java
- POO
- Collections
- Streams
- SQL
- Spring Boot
- JPA / Hibernate
- Testing
- Spring Security
- Docker
- Microservicios
- Git
- Arquitectura
- Debugging
- Optimización

## Simulaciones

- Entrevista técnica
- Code review
- Reto backend
- Debugging
- Defensa del proyecto bancario

## Proyecto

Defensa técnica del Sistema Bancario Enterprise.

El estudiante debe saber explicar:

- Arquitectura
- Capas
- DTOs
- Validaciones
- Manejo de errores
- Seguridad
- Testing
- Docker
- Microservicios
- Optimización de base de datos

## Reglas

- Responder con claridad.
- Justificar decisiones técnicas.
- Usar ejemplos del proyecto bancario.
- Explicar ventajas y desventajas.
- Pensar como desarrollador Backend profesional.

## Al finalizar

El estudiante será capaz de afrontar entrevistas técnicas Junior Java Backend con mayor seguridad.
