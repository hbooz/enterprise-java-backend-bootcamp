# Módulo 11 - Entrevistas Técnicas Java Backend

## Objetivo

Preparar al estudiante para superar entrevistas técnicas Junior y Semi Senior de Java Backend utilizando simulaciones, retos y defensa de proyectos.

---

# Perfil esperado

Al finalizar el bootcamp el estudiante será capaz de desarrollar aplicaciones Backend utilizando:

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Spring Security
- JWT
- Docker
- Docker Compose
- Spring Cloud
- Git
- GitHub
- GitFlow
- JUnit 5
- Mockito
- MockMvc
- OpenAPI (Swagger)
