# Módulo 6 - Spring Security

## Objetivo

Aprender a proteger aplicaciones Spring Boot utilizando autenticación y autorización.

## Temas

- Spring Security
- Authentication
- Authorization
- JWT
- Refresh Token
- BCrypt
- Roles
- Permisos
- Filtros
- SecurityFilterChain
- UserDetailsService
- CORS
- CSRF

## Proyecto

Seguridad del sistema bancario:

- Login
- Registro
- Autenticación
- Autorización
- Protección de endpoints

## Reglas

- Nunca guardar contraseñas en texto plano.
- Usar BCrypt.
- Proteger todos los endpoints privados.
- Validar JWT en cada petición.
- Aplicar principio de mínimo privilegio.

## Al finalizar

El estudiante será capaz de implementar autenticación y autorización profesional en Spring Boot.
