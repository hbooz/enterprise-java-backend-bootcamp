# 🚀 Enterprise Java Backend Bootcamp

Bootcamp especializado para formar un **Java Backend Developer Junior/Junior+** con un enfoque empresarial, utilizando las tecnologías y buenas prácticas más demandadas en la industria.

Este repositorio también sirve como **base de conocimiento para Google AI Studio**, permitiendo crear un profesor de IA especializado en Java Backend.

---

# 🎯 Objetivos

Al finalizar este bootcamp serás capaz de:

- Desarrollar APIs REST con Spring Boot.
- Diseñar arquitecturas Backend limpias.
- Implementar JPA e Hibernate.
- Optimizar consultas SQL.
- Aplicar SOLID y Clean Code.
- Escribir pruebas con JUnit 5 y Mockito.
- Documentar APIs con OpenAPI (Swagger).
- Implementar autenticación con JWT.
- Dockerizar aplicaciones.
- Construir microservicios con Spring Cloud.
- Trabajar con GitFlow y GitHub.
- Utilizar IA para aumentar la productividad.
- Aprobar entrevistas técnicas Java Backend.
